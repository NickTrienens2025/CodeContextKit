# ``CodeContextKitCore``

The foundational engine for architectural code analysis and surgical context packing.

## Overview

CodeContextKitCore provides the base types, utilities, and protocols used across the CodeContextKit ecosystem. It is designed to be lightweight, thread-safe, and optimized for high-performance symbol extraction and token estimation.
